Fast document scanning with an AI twist.

ScanBoy is a lightning fast document scanner. Create high quality PDFs with selectable text and quickly generate filenames using artificial intelligence so that you can move on with your day. If you want to name your own documents and avoid using AI altogether, you can do that too.

Features

Get in. Get out.

Scan and process documents quickly. Save all documents to your favorite folder.

Selectable Text

Quick optical character recognition algorithm processes all documents so you can select text later.

Templates

If you scan the same types of document often, save time naming them with filename templates.

AI Filenames

Generate short, concise filenames based on the content of your scanned document.
