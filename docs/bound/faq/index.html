Move Audio Files

Bound tries to group files based on their metadata and file names. Sometimes this file information is not very clear and files that should be put into the same audiobook get split into multiple audiobooks. Follow the below steps to merge the files back together.

  1. Tap the Edit Audiobook button.
  2. Under the Audio Files section, tap on the audio file you would like to move.
  3. Tap on on Move Audiobook.
  4. Tap the audiobook you would like to move the file into.
  5. The file has now been moved. If you like, you can reorder files by tapping and dragging the file's handle.
Reorder Audio Files

Follow the below steps to reorder the files of your audiobook.

  1. Tap the Edit Audiobook button.
  2. Under the Audio Files section, tap and drag the handle of the file you would like to reorder.
  3. Drop the file into the order you want.
  4. Tap Save to complete the process.
Download Audio Files with an Unsupported Cloud Service

If you do not see your preferred cloud service in Bound, you may still be able to use it. Many cloud services support integration with Apple’s Files app and can be used to import files directly into Bound.

  1. Download and set up the third-party cloud app.
  2. In Bound, tap the Add Audiobook button.
  3. Tap Files.
  4. Tap Browse.
  5. Tap the button at the top of the Browse screen. If you do not see the button, tap Browse again.
  6. Tap Edit.
  7. Turn on the third-party apps that you want to use in the Files app.
  8. Tap Done.
  9. Tap on the cloud service.
  10. Navigate the server’s directory structure and find your audio files.
  11. Tap files to download them individually.
  12. To download all files in a folder, tap Select then tap Select All. Finally, tap Open.
Download Audio Files with SMB

Don't want to use a cloud service? Maybe you have a network share where you store your audiobooks. This functionality is built into the Apple Files app and can be done by following the steps below.

  1. Tap the Add Audiobook button.
  2. Tap Files.
  3. Tap Browse.
  4. Tap the button at the top of the Browse screen. If you do not see the button, tap Browse again.
  5. Tap Connect to Server.
  6. Type in the server’s address. Tap Connect.
  7. Type in the server’s log in information. Tap Next.
  8. Navigate the server’s directory structure and find your audio files.
  9. Tap files to download them individually.
  10. To download all files in a folder, tap Select then tap Select All. Finally, tap Open.
